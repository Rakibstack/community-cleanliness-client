
# CleanZone

**Live Site (Client): https://clean-zone-client.web.app/

## Short Description
CleanZone is a cleanliness and complaint-management web application where users can report issues, track complaints, and contribute to clean-up efforts. The platform is fully responsive, user-friendly, and designed for smooth performance.

## Features
- Users can submit new issues/complaints with title, category, location, description, image, and suggested fix budget.
- Each complaint has a detailed page showing image, date, category, description, and a **“Pay Clean-Up Contribution”** button.
- User authentication system: logged-in users can manage, edit, or view their submitted issues.
- Search and filtering options based on category, location, date, or budget.
- Responsive UI that works perfectly on desktop, tablet, and mobile devices.
- Image upload support using hosted image URLs or cloud storage.
- Secure REST APIs (JWT/auth protected routes) for safe data communication.

## Tech Stack
- **Client:** React, React Router, Tailwind CSS
- **Server (optional):** Node.js, Express.js
- **Database (optional):** MongoDB / Firebase / PostgreSQL
- **Authentication:** JWT / Firebase Auth
- **Hosting:** Netlify / Vercel (client), Render / Railway / Heroku (server)
